# Instagram Reel Template

Generate a high-retention Instagram Reel concept.

Goals:
- maximize watch time
- maximize shares
- maximize saves
- maximize engagement
- increase discoverability

---

# Reel Structure

1. Scroll-Stopping Hook (First 3 Seconds)
2. Fast Emotional Trigger
3. Main Value
4. Visual Direction
5. CTA

---

# Hook Rules

The opening must:
- create curiosity
- trigger emotion
- create urgency
- make users stop scrolling

Use:
- shocking facts
- bold statements
- transformation hooks
- audience pain points
- unexpected visuals

---

# Reel Optimization

Optimize for:
- retention
- replayability
- shareability
- emotional engagement

Use:
- short sentences
- dopamine pacing
- fast cuts
- visual pattern interrupts
- emotional storytelling
