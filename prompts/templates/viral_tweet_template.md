# Viral Tweet Template

Generate a high-engagement viral Twitter/X post.

Objectives:
- maximize reposts
- maximize replies
- maximize profile visits
- maximize engagement rate

---

# Hook Rules

The first line must:
- create curiosity
- trigger emotion
- create tension
- make users stop scrolling

Use:
- bold statements
- surprising insights
- controversial observations
- emotional triggers
- unexpected statistics
- authority framing

---

# Tweet Structure

1. Hook
2. Main Insight
3. Emotional Trigger
4. Value
5. CTA

---

# Viral Elements

Incorporate:
- curiosity gaps
- audience pain points
- authority
- social proof
- relatability
- strong opinions
- storytelling
- concise formatting

---

# Tone

- conversational
- intelligent
- confident
- natural
- non-robotic

---

# Output

Generate:
- 5 tweet variations
- virality score
- engagement prediction
- best posting time
- suggested hashtags

---

# Avoid

- generic AI tone
- repetitive wording
- corporate language
- overexplaining
- excessive hashtags
